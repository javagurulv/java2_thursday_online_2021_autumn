Постановка задачи

Предметная область: хранение, учет и формирование
портфеля ценных бумаг для клиента

Задача: разработать электронную систему для управления
портфелем ценных бумаг.

Задача минимум:
- обширная база данных по доступным для покупки ценным
бумагам (акции и облигации);
- импорт/экспорт из базы данных;
- механизм поиска и фильтрации ценных бумаг в базе данных
по множеству параметров;
- автоматическое формирование портфеля ценных бумаг под
нужды конкретного пользователя;

Задача максимум:
- обновление рыночных цен в базе данных из интернета
(если возможно);
- учет портфелей пользователей по истечению времени
(подсчет прибыли/убытков, доходности);
- ребалансировка портфелей пользователей.
